# PRD — 직관생활 MVP

## 1. 제품 비전
**한줄 요약**: 야구 직관을 기록하고, 예쁜 카드로 만들어서, 인스타에 바로 공유하는 앱
**타겟 유저**: 20~30대 여성 KBO 팬

## 2. 완료된 기능 ✅

### 2-1. 이메일 로그인
- 이메일 매직링크 방식 (비밀번호 없음)
- Gmail SMTP 연동 (발송 안정성 확보)
- 세션 유지 (재로그인 불필요)
- 카카오 로그인: 비즈앱 전환 후 추가 예정

### 2-2. 온보딩 (최초 1회)
- 10개 구단 2열 그리드, 파스텔 색상 카드
- 팀 선택 → Supabase users 테이블 저장
- 로그인 필수 가드

### 2-3. 홈 화면
- 인사말 + 팀 이모지
- 시즌 요약 (직관 횟수, 승률 - DB 실시간)
- 최근 기록 카드 (카드 보기 버튼 포함)
- "오늘의 직관 기록하기" CTA
- 하단 탭바 (홈/기록/달력/통계/마이)

### 2-4. 직관 기록 작성
- 날짜 선택 (기본값: 오늘)
- 구장 선택 (9개 구장 카드)
- 상대팀 선택 + 스코어 입력
- 홈/원정 토글
- 한줄 코멘트
- 사진 업로드 (최대 5장, Supabase Storage)
- 저장 → DB INSERT → 카드 생성 페이지로 자동 이동

### 2-5. 인스타 카드 생성 ⭐ 킬러 기능
- 실제 DB 데이터로 파스텔 카드 렌더링
- 사진 있으면 카드에 표시
- 팀 pastel 컬러 자동 적용
- 승률/직관횟수 등 통계 자동 표시
- PNG 이미지 다운로드 (html2canvas-pro)
- Web Share API 공유 (모바일)

### 2-6. 승률 통계
- 전체 승률 (원형 표시)
- 전적 (N경기 N승 N패 N무)
- 홈/원정 승률 비교
- 최근 5경기 결과 (초록/빨강 dot)
- 상대팀별 전적 (바 차트)
- DB 실시간 계산

### 2-7. 달력 뷰
- 월간 캘린더 (월 이동 가능)
- 직관 날짜에 결과별 dot (승=초록, 패=빨강, 무=회색)
- 날짜 탭 → 기록 요약 표시
- 오늘 날짜 강조
- DB 실시간

## 3. 미구현 기능 (다음 단계)

### Phase 2: 고도화
- KBO 데이터 자동 매칭 (날짜+구장 → 경기 결과 자동)
- 클래식/볼드 카드 템플릿
- 사진 크롭/위치 조정
- 카카오 로그인
- 집관 기록
- 마이페이지 (프로필 수정, 팀 변경)

### Phase 3: 확장
- 구장 먹거리 가이드
- 소셜 기능 (친구, 승률 비교)
- PWA (오프라인, 앱 설치)
- 숏폼 영상 생성
- 유꾸마켓 연동
