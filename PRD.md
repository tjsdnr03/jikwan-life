# PRD — 직관생활 MVP

## 1. 제품 비전
**한줄 요약**: 야구 직관을 기록하고, 예쁜 카드로 만들어서, 인스타에 바로 공유하는 앱
**타겟 유저**: 20~30대 여성 KBO 팬

## 2. 완료된 기능 ✅

### 2-1. 이메일 로그인
- 이메일 OTP 코드 방식 (8자리 코드, 매직링크 아님 — 같은 페이지에서 코드 입력)
- Gmail SMTP 연동 (발송 안정성 확보)
- 세션 유지 (재로그인 불필요)
- 카카오 로그인: 사업자등록 → 비즈앱 전환 후 추가 예정 (KOE205 에러로 현재 불가)

### 2-2. 온보딩 (최초 1회)
- 10개 구단 2열 그리드, 파스텔 색상 카드
- 팀 선택 → Supabase users 테이블 저장
- 로그인 필수 가드

### 2-3. 홈 화면
- 인사말 + 팀 캐릭터(고양이)
- 시즌 요약 (직관 횟수, 승률 - DB 실시간)
- 최근 기록 카드 (카드 보기 버튼 포함)
- 오늘의 경기 카드 (KBO 일정 연동)
- "오늘의 직관 기록하기" CTA
- 하단 탭바 (홈/기록/달력/통계/마이)

### 2-4. 직관 기록 작성 / 관리
- 날짜 선택 (기본값: 오늘)
- 구장 선택 (9개 구장 카드)
- 날짜+구장 선택 시 KBO 데이터 자동 매칭 (상대팀, 스코어 자동 채움)
- 홈/원정 토글
- 한줄 코멘트
- 사진 업로드 (최대 5장, Supabase Storage)
- 저장 → DB INSERT → 카드 생성 페이지로 자동 이동
- 기록 전체 목록, 상세 보기, 수정/삭제

### 2-5. 인스타 카드 생성 ⭐ 킬러 기능
- 실제 DB 데이터로 파스텔 카드 렌더링
- 사진 있으면 카드에 표시
- 팀 pastel 컬러 자동 적용
- 승률/직관횟수 등 통계 자동 표시
- PNG 이미지 다운로드 (html2canvas-pro)
- Web Share API 공유 (모바일)

### 2-6. 승률 통계
- 전체 승률 (원형 표시)
- 전적 (N경기 N승 N패 N무)
- 홈/원정 승률 비교
- 최근 5경기 결과 (초록/빨강 dot)
- 상대팀별 전적 (바 차트)
- 상대팀 시점 통계 ("혹시 나는 상대팀 승리요정? 🧚")
- DB 실시간 계산

### 2-7. 달력 뷰
- 월간 캘린더 (월 이동 가능)
- 직관 날짜에 결과별 dot (승=초록, 패=빨강, 무=회색)
- KBO 월간 경기 일정 오버레이 (상대팀 캐릭터, 홈/원정 테두리 구분)
- 날짜 탭 → 기록 요약 표시
- 오늘 날짜 강조
- DB 실시간

### 2-8. 마이페이지
- 프로필 표시
- 응원팀 변경
- 로그아웃

### 2-9. 팀 캐릭터
- 자체 고양이 캐릭터 (AI 생성, 10팀 × 팀 컬러)
- 구단 마스코트와 겹치지 않는 디자인, 모자/나비넥타이로 팀 구분
- 배경 제거(rembg) 처리 완료

### 2-10. PWA
- manifest, 앱 아이콘 설정 완료
- 홈 화면에 추가 가능

## 3. 미구현 기능 (다음 단계, 우선순위순)

### Phase 2: 고도화 (높은 우선순위)
- 인스타 카드 디자인 개선 (사진 크롭/위치 조정, 가로/세로 대응)
- 클래식/볼드 카드 템플릿 추가
- 전체 UI 폴리싱 (여백/폰트/디자인 통일성)
- 캐릭터 이미지 스타일 통일

### Phase 2: 고도화 (중간 우선순위)
- 카카오 로그인 (사업자등록 → 비즈앱 전환 후)
- 집관 기록 기능
- 닉네임 설정/수정
- 오프라인 캐싱 (서비스 워커)

### Phase 3: 확장
- 구장 먹거리 가이드
- 소셜 기능 (친구, 승률 비교)
- 숏폼 영상 생성
- 유꾸마켓 연동

## 5. 라이선스/법적 주의사항
- KBO 구단 로고/마스코트/엠블럼 사용 절대 금지
- 구단 이름은 정보 전달 목적으로만 사용
- 선수 이름 사용 금지 (퍼블리시티권 이슈, MVP 범위)
- 유사 마스코트 캐릭터 제작 금지 (부정경쩁방지법)
- 자체 고양이 캐릭터 + 팀 색상만 활용
