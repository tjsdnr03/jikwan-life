# PRD — 직관생활 MVP

## 1. 제품 비전

**한줄 요약**: 야구 직관을 기록하고, 예쁜 카드로 만들어서, 인스타에 바로 공유하는 앱

**경쟁 차별화**:
- 기존 앱들: 기록에 집중 → SNS 공유는 스크린샷 수준
- 직관생활: 기록 + **인스타 최적화 카드 자동 생성** (킬러 기능)

**타겟 유저**: 20~30대 여성 KBO 팬 (승요/패요 문화, 인스타 공유 문화)

---

## 2. MVP 기능 명세

### 2-1. 회원가입 / 로그인
- **방법**: Supabase Auth (카카오 로그인 우선, 구글 추후)
- **플로우**: 앱 진입 → 로그인 → 최초 시 온보딩으로 이동

### 2-2. 온보딩 (최초 1회)
- **응원팀 선택**: 10개 구단 카드 그리드 (팀 컬러 + 팀명)
- **닉네임 설정**: 선택사항
- **완료 후**: 홈 화면으로 이동

### 2-3. 홈 화면
- 상단: 프로필 (닉네임 + 팀 뱃지 + 시즌 승률)
- 중앙: 최근 직관 기록 카드 3개 (가로 스크롤)
- 하단 CTA: "오늘의 직관 기록하기" 버튼
- 하단 탭: 홈 / 기록 / 달력 / 통계 / 마이

### 2-4. 직관 기록 작성 (핵심 플로우)
```
Step 1: 날짜 선택 (기본값: 오늘)
  ↓
Step 2: 구장 선택 (9개 구장 리스트)
  ↓
Step 3: 경기 자동 매칭 (KBO 데이터에서 해당 날짜+구장 경기 검색)
  → 매칭 성공: 팀명, 스코어 자동 표시
  → 매칭 실패: 수동 입력 (팀 선택 + 스코어 입력)
  ↓
Step 4: 사진 추가 (선택, 최대 5장)
  → 촬영 or 갤러리에서 선택
  → Supabase Storage에 업로드
  ↓
Step 5: 한줄 코멘트 (선택, 최대 100자)
  ↓
Step 6: 저장 → "카드 만들기" CTA 표시
```

### 2-5. 인스타 카드 생성 (킬러 기능)
```
Step 1: 템플릿 선택 (MVP: 파스텔 1종, 이후 클래식/볼드 추가)
  ↓
Step 2: 사진 선택 (기록에 첨부된 사진 중 1장)
  → 가로/세로 자동 감지, 최적 배치
  ↓
Step 3: 카드 미리보기 (9:16 비율)
  → 경기 결과, 승률, 코멘트 자동 배치
  → 팀 파스텔 컬러 자동 적용
  ↓
Step 4: 저장 / 공유
  → "이미지 저장" (갤러리에 저장)
  → "인스타 스토리 공유" (Instagram Story sharing API)
```

### 2-6. 승률 통계
- 전체 승률 (원형 차트)
- 홈/원정 승률
- 상대팀별 승률
- 월별 승률 추이

### 2-7. 달력 뷰
- 월간 캘린더
- 직관 간 날짜에 승/패 표시 (초록/빨강 dot)
- 날짜 탭 → 해당 기록 상세로 이동

## 4. 비기능 요구사항

- **성능**: 카드 생성 3초 이내
- **이미지**: 내보내기 해상도 1080x1920px (인스타 스토리 최적)
- **오프라인**: PWA 기본 캐싱 (오프라인에서 과거 기록 열람 가능)
- **반응형**: 모바일 최우선 (375px 기준), 태블릿/데스크톱 대응은 추후

## 5. MVP 제외 항목 (Phase 2 이후)

- 집관 기록

- 구장 먹거리 가이드
- 숏폼 영상 생성
- 위젯
- 선발 라인업 표시
- 업적/스탬프
- 다크모드
